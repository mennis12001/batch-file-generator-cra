
# Batch File Generator (CRA)

Create React App version of the Batch File Generator (Option A).

## Local dev
```bash
npm install
npm start
```

## Build
```bash
npm run build
```
Outputs production files to `build/`.

## Deploying to Vercel
- **Framework Preset:** Create React App
- **Build Command:** `npm run build`
- **Install Command:** `npm install`
- **Output Directory:** `build`
- **Root Directory:** repo root (where `package.json` lives)

> You do **not** need `vercel.json` for CRA.
